# 飞书默认头像生成器

纯前端 HTML5 应用，用于生成 1024×1024 的方形头像图片。用户输入两个字的名字，自由调整背景配色，实时预览并下载 JPG 格式的头像。

## 在线使用

👉 **[立即使用](https://jijjk10.github.io/avatar-generator/)**

也可以下载 `index.html`，双击在浏览器中打开即可使用，无需任何后端或服务器。

## 功能特性

- **三个背景模式**：纯色、渐变（8 个方向）、特殊配色方案（15 个预设 + 随机生成）
- **RGB 精确调色**：每个颜色通道 0-255 滑块 + 取色器 + Hex 值显示，支持背景和文字独立配色
- **实时预览**：调整任意参数即时反映在圆形预览区，所见即所得
- **高清导出**：Canvas 渲染 1024×1024 分辨率，导出为 JPG 格式
- **移动端适配**：桌面端左右分栏，移动端上下布局，横屏自动切换，触屏设备长按保存
- **暗色模式**：自动跟随系统亮色/暗色主题
- **状态持久化**：调整参数自动保存到浏览器本地存储，刷新不丢失
- **零依赖**：纯原生 HTML/CSS/JavaScript，不依赖任何框架或构建工具

## 使用方法

1. 在输入框中输入**两个字**的名字（必须两个字）
2. 选择背景模式：
   - **纯色**：拖动 RGB 滑块或使用取色器选择单一背景色
   - **渐变**：分别设置起始颜色、结束颜色和渐变方向
   - **特殊方案**：从 15 个预设渐变中选取，或点击「🎲 随机配色」
3. 在「文字颜色」区域调整字的颜色
4. 点击「⬇ 下载头像」按钮下载 1024×1024 的 JPG 图片
5. 移动端点击下载后会弹出预览图，长按图片即可保存到相册

## 技术实现

| 项目 | 说明 |
|------|------|
| 字体 | Noto Sans SC Medium (500)，通过 jsDelivr CDN 加载，国内可访问 |
| Canvas | 离屏 Canvas 渲染，预览与导出复用同一绘制函数 |
| 导出格式 | JPEG，质量 0.92，文件命名 `头像-{名字}.jpg` |
| 存储 | localStorage 自动保存配色参数 |
| 部署 | GitHub Pages，无需服务器 |

## 可调参数

以下参数在代码中以常量形式定义，可根据需要修改（无需改动 UI）：

```javascript
const FONT_SIZE = 339;          // 字号（基于 1024 参考分辨率）
const LETTER_SPACING = 5;       // 字间距（基于 1024 参考分辨率）
const FONT_WEIGHT = 500;        // 字重（500 = Medium）
const EXPORT_SIZE = 1024;       // 导出图片尺寸
const JPEG_QUALITY = 0.92;      // JPG 导出质量
```
